#  Book Catalog - Android App

Мобильное клиент-серверное приложение для учета прочитанных книг.

##  Архитектура
Проект строго следует принципам **Clean Architecture**:
* **Presentation Layer:** Jetpack Compose, ViewModels (MVI/MVVM).
* **Domain Layer:** Use Cases (сценарии использования), интерфейсы репозиториев, бизнес-модели.
* **Data Layer:** Room (локальная БД), Ktor Client (сеть), реализация репозиториев.

##  Технологический стек
* **UI:** Jetpack Compose + Material Design 3
* **Сетевые запросы:** Ktor Client Android
* **Локальная БД:** Room (SQLite) - Offline-first подход
* **Внедрение зависимостей (DI):** Dagger Hilt
* **Навигация:** Jetpack Navigation Compose
* **Тестирование:** JUnit 4 (Unit-тесты для бизнес-логики), Compose Test Rule (UI-тесты)

##  Главные фичи
* Независимая система авторизации на основе токенов (токены сохраняются локально).
* **Синхронизация данных:** Приложение мгновенно читает данные из кэша (Room) и асинхронно обновляет их с сервера.
* Динамическая панель статистики (подсчет прочитанных книг и средней оценки на лету).
* "Живой" поиск по автору и названию.
* Удобное удаление с помощью жеста **Swipe-to-Dismiss** (смахивание).
* Полный цикл создания и редактирования книг.
